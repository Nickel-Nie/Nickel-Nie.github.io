首先判断数组中1的总数量num：

  1. 如果num==0，直接返回[0,2]即可；
  2. 如果num % 3 != 0，则说明肯定无法三等分，返回[-1,-1]；
  3. 如果num % 3 == 0，仍需继续讨论，见下文。

由于前导0可以忽略，我们先确定三个部分的起始点，即以每个部分的第一个1为起始点，记录三个指针：first，second，third。影响是否相等的因素就剩下：1的位置以及后缀0的个数。

我们需要注意到：arr[third...arr.length-1]这一段（第三部分）是已经固定了的，因此可以得到其长度len = arr.length-third，这时需要去判断第一部分和第二部分的长度是否大于len，即是否满足first + len <= second && second + len <= third，如果不满足，则直接返回[-1,-1]。

如果满足，接下来就只需要判断arr[first...first+len-1], arr[second...second+len-1]，arr[third...third+len-1]中的每一位是否相同即可。
