剑指 Offer 43. 1～n 整数中 1 出现的次数

输入一个整数 n ，求1～n这n个整数的十进制表示中1出现的次数。

例如，输入12，1～12这些整数中包含1 的数字有1、10、11和12，1一共出现了5次。

思路：

n位十进制数中，1出现的次数有如下规律：

例子：

一个n位数:
$$
{a_n}{a_{n-1}}{a_{n-2}}…{a_2}{a_1}
$$
对第i位$a_i$进行分析：

（1）.向上进行分析(只看这一位)，且只看$a_i$的贡献：

  • if $a_i==0$：那么可以不看这一位数，因为它没有对1的次数增加起到作用。
  • if $a_i==1$：令它后面的数：digit = ${a_{i-1}}{a_{i-2}}…{a_1}$，$a_i$这一位所做的贡献为：digit+1(0~digit)次。
  • if $a_i>1$：那么这一位所做的贡献为：$10^{i-1}$次。

（2） 向下进行分析(看这一位后面的)：

$a_i$也表示出现了$a_i$次$0$~$10^{i-1}-1$，那么这里对1的贡献就可以通过表格来获取，即table[i-1]$*a_i$。

这里可以这么理解，因为都是0~99..9($i-1$个9)，所以第$i$位的1全部在（1）中计算得到。

比如说数字1234。

分析两个数，第四位的1和第三位的2：

(1)对于第4位的1：从1000~1234一共重复了235次，所以贡献为：235.

​ 对于第3位的2：从100~199一共重复了100次，所以贡献是: 100.

(2)对于第4位的1：前面只有1次(0~999)，所以贡献为：300.

​ 对于第3位的2：10001099, 11001199 共2次(0~99)，所以贡献为: 2*20.

综上：

第i位的数a_i所做的贡献就是上述两者相加后的结果。
