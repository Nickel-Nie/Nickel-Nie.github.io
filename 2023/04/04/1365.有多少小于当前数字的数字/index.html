1365.有多少小于当前数字的数字

给你一个数组 $nums$，对于其中每个元素 $nums[i]$，请你统计数组中比它小的所有数字的数目。

换而言之，对于每个 $nums[i]$ 你必须计算出有效的 $j$ 的数量，其中$j$ 满足 $j != i$ 且 $nums[j] < nums[i]$ 。

以数组形式返回答案。

  • 2 <= nums.length <= 500
  • 0 <= nums[i] <= 100
1. 暴力法

对于每个数字，分别遍历一次数组，记录其相应的输出值即可。

2. 计数排序(桶思想)

从题目中可以得到的信息是每个数都在范围$[0, 100]$内，可以通过一个数组$cnt$来记录每个数出现的次数，那么对于某一数字$i$来说，其输出值就是：$sum(cnt[0…i-1])$。

3.快速排序

我们也可以将数组排序，并记录每一个数在原数组中的位置。对于排序后的数组中的每一个数，我们找出其左侧第一个小于它的数，这样就能够知道数组中小于该数的数量。
